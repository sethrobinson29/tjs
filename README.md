# CHROMABLOCKS

A neon/cyberpunk falling-block puzzle game built with Phaser 3 + TypeScript.

**[Play it →](https://sethrobinson29.github.io/tjs/)**

## Controls

| Key | Action |
|-----|--------|
| ← → | Move |
| ↑ / Z | Rotate |
| ↓ | Soft drop |
| Space | Hard drop |
| Shift | Hold piece |
| P / Esc | Pause |

## Stack

- [Phaser 3](https://phaser.io/) — 2D game framework
- TypeScript
- Vite

## Development

```bash
npm install
npm run dev      # localhost with HMR
npm run build    # production build → dist/
npm run deploy   # build + push to GitHub Pages
```
